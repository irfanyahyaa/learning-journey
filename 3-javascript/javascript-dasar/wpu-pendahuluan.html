Javascript - Pendahuluan

Instruksi:


1. Untuk mencoba Javascript kita dapat mengakses console, melalui inspect element pada browser


2. Kita bisa menghapus aksi pada console dengan klik kanan, dan clear console

Note:


1. Tipe data Number pada Javascript tidak memiliki integer (bilangan bulat)



2. Dapat menyimpan angka tanpa desimal (integer), contoh: 10, 1500, 123456. Sebenernya yang diproses oleh Javascript adalah bilangan pecahan, dan tingkat akurasi dari hal tersebut hanya sampai 15 digit saja

Contoh: untuk menyimpan angka 10 kita dapat menuliskan di console, dan console pun mengembalikan angka 10 tadi. Begitu pun untuk menyimpan angka 1500, serta 123456. Akan berbeda jika, kita menuliskan lebih dari 16 digit angka. Angka yang akan ditampilkan oleh Javascript menjadi tidak akurat karena terjadi pembulatan



3. Dapat menyimpan angka desimal (float). Notasi koma untuk bilangan desimal dalam Javascript menggunakan titik (.). Kita dapat menyimpan angka di belakang koma maksimal sebanyak 17 digit

Contoh: untuk menyimpan angka 3.14 kita dapat menuliskan di console, dan console pun mengembalikan angka 3.14 tadi. Begitu pun untuk menyimpan angka 0.5, serta 100.00



4. Kita juga dapat melakukan operasi matematika pada bilangan desimal, tapi terkadang nilai nya tidak selalu akurat

Contoh: ketika kita ingin menjumlahkan nilai 0.3 dengan nilai 0.7 maka, akan menghasilkan nilai 1. Berbeda ketika kita ingin menjumlahkan nilai 0.2 dengan 0.1, hasil yang kita dapatkan yaitu 0.30000000000000004. Ada bilangan pecahan yang cukup kecil, sehingga bilangan tidak benar-benar persis seperti yang kita inginkan



5. Kita dapat pula menyimpan angka yang sangat besar dan sangat kecil dengan eksponen (notasi nya e)

Contoh: ketika kita menyimpan angka 123e5 maka Javascript akan mengembalikan angka 12300000. Begitu pun ketika kita menyimpan angka 123e-5 maka Javascript akan mengembalikan angka 0.00123. Notasi plus (+) pada eksponen menunjukkan angka nya tidak desimal, sedangkan notasi minus (-) menunjukkan angkanya bernilai desimal



6. Kita dapat pula menyimpan angka bernilai negatif



7. Jangan pernah mengawali angka dengan 0. Karena Javascript akan menerjemahkannya menjadi bilangan oktal. Begitu pula jika kita mengawali angka dengan 0x. Javascript akan memperlakukannya menjadi angka heksadesimal

Contoh: jika kita menyimpan angka 022, Javascript akan menerjemahkannya menjadi angka 18, serta jika kita menyimpan angka 0xFF, Javascript akan mengembalikan angka tersebut menjadi 255



8. Javascript juga memiliki angka spesial, yaitu Infinity, -Infinity, dan NaN (Not a Number)

Contoh: jika kita membagi angka positif dengan angka 0, maka Javascript akan mengembalikan nilai Infinity. Jika kita membagi angka negatif dengan 0 maka Javascript akan mengembalikan nilai -Infinity. Sedangkan nilai NaN akan dikembalikan oleh Javascript jika kita membagi angka 0 dengan angka 0, atau membagi suatu angka dengan string. Namun hasilnya akan berbeda jika string tersebut berisi angka, Javascript akan memperlakukan string tersebut sebagai suatu angka



9. Operator dalam Javascript merupakan simbol yang digunakan untuk melakukan operasi. Untuk melihat fungsi dari semua operator secara lengkap bisa klik disini



10. Terdapat 3 jenis operator berdasarkan jumlah operand nya, yaitu Unary (butuh 1 operand), Binary (butuh 2 operand), dan ternary (butuh 3 operand)



11. Operator Aritmatika (+, -, *, /, %). Pada operator Aritmatika, Javascript menerapkan Operator Presendence (urutan operasi matematika). Urutannya yaitu (), *, /, +, -

contoh: 1 + 2 * 3 - 4 akan menghasilkan 3, karena mengikuti aturan Operator Presendence. Jika ingin melakukan operasi berurut dari kanan ke kiri maka (1 + 2) * 3 - 4 akan menghasilkan 5



12. Operator Assignment (=, +=, -=, *=, /=, %=). Fungsi dari Assignment (=) berfungsi untuk meng-assign suatu nilai kepada variable. Ketika sudah melakukan Assignment, maka komputer kita akan mengingat aksi tersebut, meskipun kita melakukan Clear Console

contoh: var x yaitu kita mendeklarasikan sebuah variable bernama x, kemudian x = 10 yaitu kita menugaskan operand yang sebelah kanan (10) ke dalam operand yang sebelah kiri (x). Maka, ketika kita memanggil variable x, Javascript akan menampilkan nilai 10. Jika kita mendeklarasikan kembali variable x, maka nilai lama (10) akan digantikan dengan nilai barunya



13. Operator Comparison (==, !=, ===, !==, >, <, >=, <=). Operator comparison akan menghasilkan nilai boolean (true atau false)

contoh: ketika kita menjalankan perintah 10 == 10, maka akan menghasilkan nilai true. Untuk membandingkan nilai beserta tipe data nya kita bisa menggunakan operator === dan !==. Ketika kita menjalankan perintah 10 === "10", maka akan menghasilkan nilai false karena kedua nilai tersebut memiliki tipe data yang berbeda



14. Operator Logical (&&, ||, !), arti dari masing-masing operator secara berurut yaitu AND, OR, dan NOT

contoh: ketika kita ingin menjalankan perintah (x % 2 == 0) && (x < 10) dengan nilai x = 8, maka javascript akan mengembalikan nilai true. Karena, kedua ekspresi diatas bernilai true. Adapun, fungsi dari operator ! adalah membalikkan nilai suatu ekspresi, misal kita menjalankan perintah x < 20 dengan nilai x = 10 menghasilkan nilai true. Namun, jika kita menambahkan operator ! sebelum ekspresi nya maka hasilnya adalah false



15. Operator String (+), berfungsi untuk menggabungkan 2 operand dengan tipe data string

contoh: ketika kita ingin menjalankan perintah "apple" + " " + "pen", maka javascript akan mengembalikan nilai "apple pen". Kita juga bisa menggabungkan tipe data string dan number, misal kita ingin menjalankan perintah 10 + 10 + "10" akan menghasikan nilai "2010". Berbeda ketika kita ingin menjalankan perintah "10" + 10 +10, maka javascript akan mengembalikan nilai "101010"



16. Operator Typeof (typeof(operand)), berfungsi untuk mengecek tipe data dari suatu variable

contoh: kita bisa mengecek tipe data 10 dengan menjalankan perintah typeof(10), maka javascript akan mengembalikan nilai number, sedangkan jika kita menjalankan perintah typeof("10"), maka javascript akan mengembalikan nilai string



17. Operator Conditional (Condition) ? Value1 (when true) : Value2 (when false), merupakan shorthand dari ekspresi if-else

contoh: kita bisa mengecek apakah suatu bilangan itu genap atau ganjil dengan menjalankan perintah (x & 2 == 0) ? "genap" : "ganjil". Jika x = 10, maka nilai yang akan dikembalikan oleh javascript yaitu "genap", karena nilai modulus dari 10 : 2 = 0, sehingga kondisinya true dan mengembalikan nilai yang ada di sebelah kiri



18. Tipe data String pada Javascript digunakan untuk merepresentasikan data tekstual (plain text). Kita bisa membuat data bertipe string dengan membungkus data atau nilai kita dengan (" ") atau (' '). Agar data bertipe String yang kita miliki aman dari error, sebaiknya data tersebut menggunakan escape character jika memiliki karakter ("") atau (''). Kita bisa mengecek seluruh escape character dengan klik disini

contoh: kita bisa menjalankan perintah "ini adalah hari jum\'at", maka javascript akan mengembalikan nilainya menjadi "ini adalah hari jum'at"



19. Tipe data Boolean merupakan tipe data yang digunakan untuk merepresentasikan logika true atau false. Boolean biasa digunakan untuk statement pengkondisian, menentukan aksi yang berbeda, dan mengatur alur kendali program. Javascript dapat mengembalikan nilai Boolean dengan menggunakan fungsi dari Javascript itu sendiri, yaitu Boolean(nilai). Dalam tipe data Boolean terdapat nilai Truthy yang mengembalikan nilai true dan Falsy yang mengembalikan nilai false. Kita bisa mengecek nilai Truthy di link berikut, dan nilai Falsy di link berikut



20. Variable merupakan sebuah 'tempat/wadah' yang memiliki nama, yang digunakan untuk menyimpan suatu 'nilai'. Dalam suatu Variable terdapat 3 proses yang kita lakukan yaitu Deklarasi, Inisialisasi, dan Assignment. Deklarasi merupakan cara kita mendaftarkan variable ke dalam lingkup yang sesuai, Inisialisasi sendiri yaitu cara komputer untuk menyediakan memori untuk variable yang sudah kita deklarasikan, sedangkan Assignment yaitu proses menetapkan nilai yang spesifik ke dalam variable. Dalam Javascript, terdapat 3 cara untuk mendeklarasikan sebuah variable yaitu var, let, dan const. Ketika mendeklarasikan sebuah variable kita tidak bisa mengawalinya dengan angka, sepert var 1boleh, begitu juga kita tidak diperbolehkan untuk membuat variable dengan menggunakan Keyword & Reserve Word. Untuk melihat Keyword & Reserve Word secara lengkap bisa klik link berikut

contoh: var x; merupakan proses Deklarasi dan Inisialisasi, kemudian kita mendeklarasikan x = 20; yang merupakan proses Assignment. Kita juga dapat menyingkat penulisan dari sebuah variable dengan syntax berikut
var nama = "Irfan"
